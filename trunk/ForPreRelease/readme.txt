VPSERVER 2.7 (MAX_L=151 NoExtensions)
Copyright (c) 2009 Ivanov Viktor

Простой мини-HTTP-сервер (в целом, соответсвует RFC822, RFC1123, RFC1738, RFC1808, RFC1900, RFC2068, RFC2616, RFC2817, но не включена поддержка If-* и ещё некоторых фичей).
Поддерживает n-ное колличество виртуальных хостов
ТОЛЬКО ДЛЯ СТАТИЧЕСКИХ СТРАНИЦ

Необходимые файлы для нормальной работы:
1. Server.exe - сама программа
2. mime.types - типы mime
3. config.conf - параметры. При отсутствии создаётся
4. error400.html - ошибка 400
5. error404.html - ошибка 404
6. error405.html - ошибка 405
7. error416.html - ошибка 416
8. error501.html - ошибка 501
9. error505.html - ошибка 505
10. localhost\index.html - виртуальный хост localhost с домашней страницей

Внимание! Программу завершать следует только с помощью команды QUIT, чтобы избежать повреждение логов

Можно пользоваться командой SHOW - показать список потоков (SHOWALL выводит весь список)

Команда SMODE переключает отладочный режим S.

Команда LLEV изменяет уровни лога на экране и в файле
LLEV [CLL FLL]
 CLL - в консоли
 FLL - в файле

Команда CONSOLEV отключает/включает отображение лога на экране

Для аварийного перезапуска используйте RESETT

К программе можно подключить один плагин. SDK смотри в SDK.txt

Минимальные системные требования:
Microsoft Windows 9x, NT
15MB RAM (только для программы)
Winsock version 2.02
Internet Explorer 5
20MB HD (программа + лог)
